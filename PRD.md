## Informações do Projeto

### Nome do Projeto

Agendador de Posts para Redes Sociais

### Stack Tecnológica

#### Frontend

* HTML5
* Bootstrap
* jQuery

#### Backend

* Django

#### Banco de Dados

* Neon (PostgreSQL Serverless)

## Escopo do Sistema

O sistema permitirá que usuários gerenciem e agendem publicações para múltiplas redes sociais a partir de uma única interface.

### Funcionalidades esperadas

* Cadastro e autenticação de usuários
* Gestão de perfis de redes sociais
* Criação de posts
* Agendamento de publicações
* Calendário de publicações
* Edição e cancelamento de agendamentos
* Histórico de publicações
* Dashboard com métricas básicas
* Controle de permissões e usuários
* Logs de operações
* Notificações de falhas e sucesso de publicação
